# 🎮 网页版五子棋游戏

一个精美的网页版五子棋游戏，使用原生 HTML、CSS 和 JavaScript 开发。

## ✨ 功能特性

- 🎯 标准 15x15 棋盘
- 🎨 精美的渐变UI设计
- 👥 **双人对战模式**：与好友面对面对战
- 🤖 **人机对战模式**：挑战智能AI对手
- 🎚️ **三档难度选择**：简单、中等、困难
- 🧠 智能AI算法，基于棋型评估
- ⚫⚪ 黑白双方轮流对战
- 🎮 鼠标悬停预览功能
- 🏆 自动判断胜负
- 🔄 重新开始游戏
- 📱 响应式设计，支持移动设备
- ✨ 流畅的动画效果

## 🚀 如何使用

1. 直接在浏览器中打开 `index.html` 文件即可开始游戏
2. 或者使用本地服务器运行：

```bash
# 使用 Python 3
python3 -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js (需要先安装 http-server)
npx http-server
```

3. 在浏览器中访问 `http://localhost:8000`

## 🎲 游戏规则

1. **选择模式**：开始时选择双人对战或人机对战模式
2. **黑棋先行**：游戏开始时黑棋先走
3. **轮流落子**：双方轮流在棋盘交叉点上落子
4. **获胜条件**：横、竖、斜任意方向连成五个同色棋子即获胜
5. **平局判断**：棋盘下满且无人获胜则判定为平局

### 人机对战说明
- 玩家执黑棋，AI执白棋
- **三个难度等级**：
  - 😊 **简单**：适合新手，AI会随机下棋，偶尔使用最佳策略
  - 😐 **中等**：有一定挑战性，AI攻防平衡
  - 😤 **困难**：高手对决，AI更重视防守和多步预判
- AI会自动思考并落子
- AI基于智能算法，具有一定难度
- AI会识别多种棋型：连五、活四、冲四、活三等

## 🎯 操作说明

- **选择模式**：打开游戏后先选择双人对战或人机对战
- **选择难度**（人机对战）：选择人机对战后，可以选择简单、中等、困难三个难度等级
- **落子**：点击棋盘交叉点即可落子
- **预览**：鼠标悬停在空白交叉点上可查看落子预览
- **重新开始**：点击"重新开始"按钮可开始新游戏
- **切换模式**：点击"切换模式"按钮返回模式选择界面
- **再玩一局**：游戏结束后点击弹窗中的按钮可继续游戏

## 📝 开发说明

游戏使用面向对象的方式开发，主要包含以下核心功能：

### 基础功能
- `drawBoard()`: 绘制棋盘和网格
- `drawPiece()`: 绘制棋子
- `handleClick()`: 处理落子逻辑
- `checkWin()`: 检查是否有玩家获胜
- `restart()`: 重新开始游戏

### AI算法
- `getBestMove()`: 获取AI的最佳落子位置（根据难度调整策略）
- `selectBestMove()`: 从候选位置中选择最佳位置
- `evaluatePosition()`: 评估某个位置的得分（根据难度调整攻防权重）
- `getScoreForPlayer()`: 计算玩家在某位置的得分
- `getLine()`: 获取某个方向上的棋型
- `evaluateLine()`: 评估棋型并返回得分

### AI难度机制
- **简单难度**：
  - 只有25%的概率选择最佳位置
  - 75%的概率随机选择附近位置
  - 防守权重降低（0.5倍）
  
- **中等难度**：
  - 总是选择最佳位置
  - 攻防平衡（防守权重1.1倍）
  
- **困难难度**：
  - 总是选择最佳位置
  - 更重视防守（防守权重1.3倍）
  - 能更好地预判玩家的攻势

### AI棋型识别
- **连五** (100000分): 已经获胜
- **活四** (10000分): 下一步必胜
- **冲四** (5000分): 可以形成五连
- **活三** (1000分): 可以形成活四
- **眠三** (500分): 可以形成冲四
- **活二** (100分): 基础棋型
- **眠二** (50分): 基础棋型
